# Fake Review Detector

AI-powered Fake Review Detector using Machine Learning and NLP.

## Features
- Detect fake vs genuine reviews
- Uses TF-IDF and Logistic Regression
- Simple CLI-based system

## How to Run
1. Install libraries:
   pip install pandas numpy scikit-learn

2. Run:
   python fake_review_detector.py
